⚡ Exercise 3 – Custom Directive

The appInputFormat directive listens to the blur event (when you click away) and converts the text to UPPERCASE.

Try it out

Last converted value: {{ lastValue }}

How it works

1. Generate directive:

ng generate directive input-format

2. Implement HostListener in input-format.ts:

@HostListener('blur')
onBlur(): void {
  const val = this.el.nativeElement.value;
  this.el.nativeElement.value = val.toUpperCase();
}

3. Apply directive as HTML attribute:

<input appInputFormat />
